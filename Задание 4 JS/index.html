Создаётся массив n-случайных чисел в указанном диапазоне (количество элементов и диапазон задаются с помощью диалоговых окон). Выводится значение в виде таблицы. Выводится массив отсортированных элементов массива по возрастанию.

У каждого дня недели свой цвет в Таиланде. Вводится номер дня недели. Выводится название этого дня в прямоугольнике соответствующего цвета.
